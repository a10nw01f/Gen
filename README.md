# Gen

A c++20 arbitrary compile time execution and code generation library. 
Currently only supports msvc with visual studio.


# Dependencies
[Detours](https://github.com/microsoft/Detours)


# How to use the example
1. Clone/download the repository
2. Build the DllInjector and Hook projects
3. Run inject_dll.bat
4. Build the Example project


For more information check this post:
https://a10nw01f.github.io/generative_cpp/
